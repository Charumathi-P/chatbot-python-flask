# Chatbot using Python & Flask
An AI-powered chatbot built with Python and Flask that provides real-time conversations through a modern web interface.

##  Features
- Responsive UI
- Real-time chat
- Flask backend
- Clean and modular code
- Easy to customize
- Mobile-friendly

##  Technologies Used
- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Bootstrap

##  Project Structure
chatbot-python-flask/
│── static/
│── templates/
│── app.py
│── chatbot.py
│── requirements.txt
│── README.md

##  Installation
git clone https://github.com/yourusername/chatbot-python-flask.git
cd chatbot-python-flask
pip install -r requirements.txt
python app.py

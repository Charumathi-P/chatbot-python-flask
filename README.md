# Chatbot using Python & Flask
An AI-powered chatbot built with Python and Flask that provides real-time conversations through a modern web interface.

##  Features
- Responsive UI
- Real-time chat
- Flask backend
- Clean and modular code
- Easy to customize
- Mobile-friendly

- JavaScript
- Bootstrap
